#Quick install
```bash
git clone https://github.com/Emmanuel-Rods/JS-Email-Crawler
cd JS-Email-Crawler
npm install
```

#Setting up
open up the cookies.js file and input your credentials
```js
//credentiales
const EMAIL_OR_USERNAME = ''
const YOUR_PASSWORD = ''
```
then 
```js
npm run cookies
```
once the cookies are set in place , you can run the index.js file
